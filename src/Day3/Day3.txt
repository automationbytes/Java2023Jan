array
-----

same datatype (homogeneous elements)
insertion order (index)
dynamic access is possible
storing duplicate value is possible

Adv
---
easy to store and manipulate large datasets
random access possible

Disadv
-----
Array are fixed in size (increase/decrease)
it can store only single datatype
